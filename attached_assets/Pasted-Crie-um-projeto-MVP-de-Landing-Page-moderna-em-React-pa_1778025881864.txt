Crie um projeto MVP de Landing Page moderna em React para uma marcenaria chamada “Trindade Engenharia e Serviços”.

Objetivo: Desenvolver uma landing page profissional, elegante e estratégica para apresentar a empresa a um cliente final. A página deve valorizar a marcenaria como uma solução premium para arquitetos, designers de interiores, construtoras e clientes que buscam móveis planejados de alto padrão.

Contexto da marca: A Trindade Engenharia e Serviços atua com marcenaria sob medida, projetos personalizados e execução de móveis planejados. O posicionamento deve transmitir confiança, sofisticação, precisão técnica, acabamento de qualidade e parceria estratégica com arquitetos.

Público-alvo principal: Arquitetos e profissionais de interiores que precisam de uma marcenaria parceira para executar projetos com qualidade, prazo e atenção aos detalhes.

Observações importantes:

Este projeto é um MVP para apresentação ao cliente.
Usar textos persuasivos e com tom premium.
Usar dados fictícios onde faltarem informações reais.
Facilitar a troca futura de imagens, telefone, e-mail, endereço, depoimentos e links.
Não criar uma página genérica. A landing page deve parecer feita especificamente para uma marcenaria premium voltada a arquitetos.
